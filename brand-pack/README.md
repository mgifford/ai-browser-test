# AI Browser Test Brand Pack

This pack contains the visual and narrative DNA for the AI Browser Test project.

## Adoption Steps
1. **Styles**: Copy `tokens.css` into your CSS directory and `@import` it.
2. **Components**: Use the specifications in `components.md` to build new UI. Ensure the CSS classes match the variable names.
3. **Voice**: Review `voice.md` before writing new technical guides or UI copy.
4. **AI/LLM Config**: Provide `ai-instructions.md` as a System Prompt when using LLMs to generate PRs or feature updates for this repo.

## Visual Fingerprint
- **Light**: Parchment (#fdfaf3)
- **Dark**: Forest (#0a1a12)
- **Accent**: Coral (#ff6b6b)
